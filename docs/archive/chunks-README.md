# Agent-Ready Chunks

This directory contains small, implementation-ready work chunks derived from
`docs/archive/backlog.md`. Use these when a backlog slice is too broad to hand directly
to Codex, Claude, or another implementation agent.

Each chunk should name:

- the goal and non-goals,
- the relevant `spec.md` sections,
- ADR dependencies,
- likely production files,
- tests to add or update,
- acceptance criteria,
- verification commands.

- [`003-atomic-manifest-writes.md`](../projects/atomic-manifest-writes/chunk.md)

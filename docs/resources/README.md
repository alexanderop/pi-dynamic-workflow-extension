# Resources — Material to Learn From

Reference and teaching material. Read to understand the system; not a to-do list
and not a contract. Doesn't get "finished" or archived — it just stays useful.

- [`learning/`](./learning/) — staged learning guide (00–10), glossary,
  exercises, and references for onboarding to the dynamic-workflow feature.
- [`blog/`](./blog/) — narrative write-up of how the feature works.
- [`examples/`](./examples/) — runnable example scripts (e.g. resume-cache playground).
- [`assets/`](./assets/) — images used by the docs.
